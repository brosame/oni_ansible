= Überblick =
Diese Rolle betreibt das Opennet Mediawiki.

Enthalten sind:
* Paketinstallation (MediaWiki, MySQL)
* Anlegen der Datenbank, Einrichtung automatisches Backup
* Installieren von MediaWiki Erweiterungen
* Aktivieren von MediaWiki Erweiterungen
* Übertragung der Konfiguration

= Konfiguration =

== Erweiterung ContactPage ==
Für das Kontaktformular muss ein Wiki Benutzer "OpennetContactForm" (admin@opennet-initiative.de angelegt werden. Die Verifikation der E-Mail Adresse ist erforderlich. Anschlißend funktioniert das Kontaktformular.

== Dynamische Sidebar ==
Betrifft MediaWiki:Sidebar - Anzeige von Änderungen nur bei angemeldeten Nutzern:
{{#ifanon:Mitmachen|recentchanges-url{{!}}Änderungen}} 

== Alternative Startseite ==
Nutze die "Move" Funktion und setze unter MediaWiki:Mainpage die neue Startseite.
